# 🦸‍♂️ ¿Qué Personaje Eres? - iOS SwiftUI App

![iOS](https://img.shields.io/badge/iOS-000000?style=for-the-badge&logo=ios&logoColor=white)
![Swift](https://img.shields.io/badge/Swift-FA7343?style=for-the-badge&logo=swift&logoColor=white)
![SwiftUI](https://img.shields.io/badge/SwiftUI-0059E3?style=for-the-badge&logo=apple&logoColor=white)
![Xcode](https://img.shields.io/badge/Xcode-1575F9?style=for-the-badge&logo=xcode&logoColor=white)

Una aplicación móvil nativa e interactiva para iOS que, mediante un cuestionario dinámico de opciones múltiples, analiza la personalidad del usuario y determina con qué personaje icónico (ej. Tony Stark / Iron Man) comparte más similitudes.

Este proyecto fue desarrollado como parte del entregable **Reto 5** para la carrera de Ingeniería en Sistemas Computacionales de la UVEG, demostrando la transición y el dominio del desarrollo declarativo en el ecosistema de Apple.

---

## ✨ Características Principales

* **Interfaz Declarativa (SwiftUI):** Construcción de vistas modernas y reactivas utilizando las últimas tecnologías de Apple, dejando atrás los Storyboards tradicionales.
* **Diseño UI/UX Inmersivo:** Implementación de fondos con gradientes de color (`LinearGradient`), esquinas redondeadas, sombras y una paleta de colores cuidadosamente seleccionada para una experiencia de usuario premium.
* **Lógica Dinámica:** Uso del manejador de estado `@State` para actualizar la interfaz en tiempo real conforme el usuario avanza por las preguntas (ej. de la Pregunta 1 a la 5).
* **Algoritmo de Selección:** Sistema de evaluación que rastrea las respuestas seleccionadas y calcula el resultado final para mostrar el personaje correspondiente junto con una breve descripción de su personalidad.

---

## 🛠️ Tecnologías y Herramientas Utilizadas

* **Lenguaje:** Swift 5+
* **Framework UI:** SwiftUI
* **Entorno de Desarrollo (IDE):** Xcode
* **Arquitectura:** Componentes reactivos basados en el estado (State-driven UI)
* **Entorno de Pruebas:** iOS Simulator

## 🚀 Instalación y Ejecución

Si deseas clonar y probar este proyecto en tu propio entorno de desarrollo (requiere macOS y Xcode):

1. Clona este repositorio en tu máquina local:
   `git clone https://github.com/Jaccstudios/Reto5-App-Que-Personaje-Eres.git`
2. Abre la carpeta del proyecto y ejecuta el archivo `.xcodeproj` con Xcode.
3. En la esquina superior izquierda de Xcode, selecciona un simulador reciente (ej. iPhone 15 Pro).
4. Presiona el botón de **Play** (▶️) o usa el atajo `Cmd + R` para compilar y ejecutar la aplicación.
